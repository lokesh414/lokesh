There are three ways in creating string. they are:
1.You can create String objects with String literal
 example:
String str="Hello!";

2.This is the common way to create a String object in java.
 example
String str1= new String("Hello!");

3.You could also convert character array into String here
 example
char ch[]={ 'H','e','l','l','o','!',};
String str1=new String(ch);
